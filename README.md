web_Canina   CANIN

DESCRIPCION DEL PROYECTO

Landing Page desarrollada para un emprendimiento de servicios caninos llamado CANIN. 
La página permite mostrar distintos servicios orientados al cuidado de mascotas, como guardería, paseos, entrenamiento y atención veterinaria.


PROBLEMATICA

Muchos dueños de mascotas no cuentan con el tiempo necesario para cuidar adecuadamente a sus perros. 
Esta página busca ofrecer una solución mostrando servicios confiables y accesibles para el bienestar de las mascotas.

TECNOLOGIAS UTILIZADAS

HTML5 (estructura semántica)
CSS3 (estilos y diseño)
Bootstrap (framework para estilos y componentes)
Git y GitHub (control de versiones)
